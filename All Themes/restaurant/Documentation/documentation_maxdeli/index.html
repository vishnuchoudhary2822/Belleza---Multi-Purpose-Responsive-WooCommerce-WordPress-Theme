About The Theme

Maxdeli theme is an advanced and multi-purpose theme for WordPress. Please read this guide thoroughly to get started with maxdeli theme. This documentation is strictly theme-related and does not cover general WordPress functionality. You need to be familiar with the WordPress platform in order to get the most out of this theme. For any general WordPress info please refer to their official documentation and forum.

WordPress Documentation | WordPress Forum

Before you install this theme:

1. Read the complete instruction
2. Make sure you have installed and running the correct/stable/latest version of WordPress
3. It is a good practice to backup your system

We recommend to use this theme on a new, clean installation or existing site without custom core modifications.

System Requirements


Required PHP libraries / modules
----------------------------------------------------------
  1. Curl
  2. ZIP
  3. Zlib
  4. GD Library
  5. Mcrypt
  6. Mbstrings
  7. Xml

Required PHP Settings (PHP.ini)
----------------------------------------------------------
  1. memory_limit = 256M
  2. upload_max_size = 64M
  3. post_max_size = 64M
  4. upload_max_filesize = 64M
  5. max_execution_time = 300
  6. max_input_time = 1000
